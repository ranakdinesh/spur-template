# MODULE CONTRACT — HOW TO BUILD ANY MODULE

> Every module — identity, workforce, reading, cat, proctoring, AI agent —
> follows this exact pattern. Learn it once, apply it forever.
> This contract is what makes the platform extensible without entropy.

---

## THE MODULE INTERFACE

Every module implements exactly this:

```go
type Module interface {
    Mount(infra *platform.Infra) error
    Definition() platform.ModuleDefinition
    HealthCheck(ctx context.Context) error
}
```

`Mount()` is called once at startup. It must:
1. Run SQL migrations
2. Extract module config from global config
3. Wire repository adapters (inject `infra.DB`)
4. Wire services (inject port interfaces)
5. Register HTTP routes on `infra.HTTP`
6. Register gRPC services on `infra.GRPC` (if any)
7. Return error on failure — never call `os.Exit`

## MODULE DIRECTORY STRUCTURE

```
modules/[name]/
├── module.go              ← Entry point. The ONLY file platform imports.
├── config.go              ← Module-specific env vars as a struct.
├── core/
│   ├── domain/            ← Pure Go structs with domain methods. Zero imports from adapters.
│   │   ├── [entity].go    ← e.g. job.go, worker.go, test.go
│   │   └── errors.go      ← Domain-specific error vars
│   ├── ports/
│   │   ├── ports.go       ← All interfaces: repos, services, commands, results
│   │   └── events.go      ← Domain events (if any)
│   └── services/
│       ├── [name]_service.go    ← One file per service
│       └── [name]_service_test.go
├── adapters/
│   ├── postgres/
│   │   ├── store.go       ← Store struct, RunInTx
│   │   ├── sqlc/          ← Generated by sqlc — DO NOT EDIT
│   │   └── [entity]_repo.go  ← One file per aggregate root
│   ├── http/
│   │   ├── routes.go      ← RegisterRoutes(r, services...)
│   │   └── handlers/
│   │       └── [name].go  ← One handler file per domain area
│   └── grpc/              ← Only if module exposes gRPC
│       ├── server.go
│       └── [name].proto
└── sql/
    ├── migrations/
    │   ├── embed.go       ← //go:embed *.sql
    │   └── NNNN_[name].sql
    └── queries/
        └── [name].sql     ← sqlc input files
```

## module.go TEMPLATE

```go
package [name]

import (
    "context"
    "embed"
    "fmt"

    "github.com/ranakdinesh/citual/platform"
    "[module]/adapters/http"
    "[module]/adapters/http/handlers"
    "[module]/adapters/postgres"
    "[module]/core/services"
    "[module]/sql/migrations"
)

// Module is the entry point. Platform imports only this.
type Module struct {
    // Exported port interfaces only — no concrete types
    [Name]Service ports.[Name]Service
}

func New() platform.Module {
    return &Module{}
}

func (m *Module) Definition() platform.ModuleDefinition {
    return platform.ModuleDefinition{
        Name:       "[name]",
        Version:    "1.0.0",
        HTTPPrefix: "/[name]",
        Permissions: []string{
            "[name].[entity].create",
            "[name].[entity].read",
            "[name].[entity].update",
            "[name].[entity].delete",
        },
    }
}

func (m *Module) Mount(infra *platform.Infra) error {
    // 1. Run migrations
    if err := infra.Migrations.Run(
        context.Background(),
        "[name]",
        migrations.FS,
    ); err != nil {
        return fmt.Errorf("[name] migrations: %w", err)
    }

    // 2. Module config
    cfg := ConfigFrom(infra.Config)

    // 3. Wire repos (concrete adapter, satisfies port interfaces)
    store := postgres.NewStore(infra.DB)

    // 4. Wire services (take port interfaces, return service impls)
    svc := services.New[Name]Service(store, infra.Log)
    m.[Name]Service = svc

    // 5. HTTP routes
    h := handlers.New[Name]Handler(svc)
    infra.HTTP.Mount(m.Definition().HTTPPrefix, http.RegisterRoutes(h))

    return nil
}

func (m *Module) HealthCheck(ctx context.Context) error {
    // Check DB connectivity for this module's tables
    return nil
}
```

## SQL MIGRATIONS TEMPLATE

```sql
-- modules/[name]/sql/migrations/0001_[name]_init.sql
-- Convention: schema = public, prefix table names with nothing
-- (public schema is the app schema — no prefix needed)

CREATE TABLE IF NOT EXISTS [entities] (
    id          UUID        PRIMARY KEY DEFAULT gen_random_uuid(),
    tenant_id   UUID        NOT NULL,   -- soft ref to identity.tenants(id)
    created_by  UUID        NOT NULL,   -- soft ref to identity.users(id)
    -- your columns
    created_at  TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at  TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

-- Always index tenant_id — every query will filter by it
CREATE INDEX idx_[entities]_tenant ON [entities](tenant_id);

-- Trigger for updated_at
CREATE OR REPLACE FUNCTION update_updated_at()
RETURNS TRIGGER AS $$
BEGIN NEW.updated_at = NOW(); RETURN NEW; END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER [entities]_updated_at
    BEFORE UPDATE ON [entities]
    FOR EACH ROW EXECUTE FUNCTION update_updated_at();
```

## PORTS (INTERFACES) TEMPLATE

```go
// modules/[name]/core/ports/ports.go

package ports

import (
    "context"
    "github.com/google/uuid"
    "[module]/core/domain"
)

// ─── Repository Interfaces ────────────────────────────────────────────────

type [Entity]Repo interface {
    Create(ctx context.Context, e *domain.[Entity]) (*domain.[Entity], error)
    GetByID(ctx context.Context, id uuid.UUID, tenantID uuid.UUID) (*domain.[Entity], error)
    List(ctx context.Context, tenantID uuid.UUID) ([]*domain.[Entity], error)
    Update(ctx context.Context, e *domain.[Entity]) error
    Delete(ctx context.Context, id uuid.UUID, tenantID uuid.UUID) error
}

// ─── Service Interfaces ───────────────────────────────────────────────────

type [Name]Service interface {
    Create(ctx context.Context, cmd Create[Entity]Cmd) (*domain.[Entity], error)
    Get(ctx context.Context, id uuid.UUID, tenantID uuid.UUID) (*domain.[Entity], error)
    List(ctx context.Context, tenantID uuid.UUID) ([]*domain.[Entity], error)
    Update(ctx context.Context, id uuid.UUID, tenantID uuid.UUID, cmd Update[Entity]Cmd) error
    Delete(ctx context.Context, id uuid.UUID, tenantID uuid.UUID) error
}

// ─── Commands and Results ─────────────────────────────────────────────────

type Create[Entity]Cmd struct {
    TenantID  uuid.UUID `json:"tenant_id"`
    CreatedBy uuid.UUID `json:"created_by"`
    // module-specific fields
}

type Update[Entity]Cmd struct {
    // module-specific fields
}
```

## HANDLER TEMPLATE

```go
// modules/[name]/adapters/http/handlers/[name].go

package handlers

import (
    "encoding/json"
    "net/http"

    "github.com/go-chi/chi/v5"
    "github.com/google/uuid"
    "github.com/ranakdinesh/citual/platform"
    "[module]/core/ports"
)

type [Name]Handler struct {
    svc ports.[Name]Service
}

func New[Name]Handler(svc ports.[Name]Service) *[Name]Handler {
    return &[Name]Handler{svc: svc}
}

func (h *[Name]Handler) Create(w http.ResponseWriter, r *http.Request) {
    // 1. Auth check — never re-parse JWT, use platform helpers
    if !platform.HasPermission(r.Context(), "[name].[entity].create") {
        http.Error(w, "forbidden", http.StatusForbidden)
        return
    }

    // 2. Parse request
    var cmd ports.Create[Entity]Cmd
    if err := json.NewDecoder(r.Body).Decode(&cmd); err != nil {
        http.Error(w, "bad request", http.StatusBadRequest)
        return
    }

    // 3. Inject auth context — never trust request body for these
    cmd.TenantID  = platform.TenantID(r.Context())
    cmd.CreatedBy = platform.UserID(r.Context())

    // 4. Call service
    result, err := h.svc.Create(r.Context(), cmd)
    if err != nil {
        // TODO: map domain errors to HTTP status codes
        http.Error(w, err.Error(), http.StatusInternalServerError)
        return
    }

    // 5. Respond
    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(http.StatusCreated)
    json.NewEncoder(w).Encode(result)
}
```

## CROSS-MODULE RULES

**RULE 1: Modules never import each other.**
```
// FORBIDDEN
import "github.com/ranakdinesh/citual/modules/identity/core/services"

// ALLOWED
// Use platform.UserID(ctx) to get identity data
// Use infra.Registry to resolve registered services
```

**RULE 2: Application tables never FK to identity schema.**
```sql
-- FORBIDDEN
FOREIGN KEY (user_id) REFERENCES identity.users(id)

-- ALLOWED
user_id UUID NOT NULL  -- application enforces the relationship
```

**RULE 3: Application modules read identity from context, never DB.**
```go
// FORBIDDEN
user, err := identityDB.GetUser(ctx, userID)

// ALLOWED
userID   := platform.UserID(ctx)
tenantID := platform.TenantID(ctx)
```

**RULE 4: Permissions are namespaced by module.**
```
identity.*      → auth module
workforce.*     → NILEX workforce module
exam.*          → exam platform
agent.*         → agentic AI module
```

```bash

cd nilex

mkdir -p modules/matching/core/{domain,ports,services}

mkdir -p modules/matching/sql/{migrations,queries}

# 3. Register in cmd/main.go
# Add: workforce.New(), matching.New()

# 4. Add to sqlc.yaml
# Add workforce block, matching block

# 5. Add to platform/config/config.go

# 6. Add theme

# Set NEXT_PUBLIC_THEME=nilex
# Replace logo asset

# 7. Generate sqlc

# 8. Build
go build ./...
```

When ready to add an AI agent module:

```
modules/agent/
├── module.go
├── config.go           ← AGENT_OPENAI_KEY, AGENT_MODEL, etc.
├── core/
│   ├── domain/
│   │   ├── task.go     ← AgentTask, TaskStatus, TaskResult
│   │   └── memory.go   ← ConversationMemory, VectorEntry
│   ├── ports/
│   │   ├── ports.go    ← AgentService, MemoryStore, ToolRegistry
│   │   └── tools.go    ← Tool interface (function calling)
│   └── services/
│       ├── agent_service.go    ← LangGraph or custom graph runner
│       ├── memory_service.go   ← Embeddings + pgvector retrieval
│       └── tool_registry.go    ← Register tools from other modules
├── adapters/
│   ├── openai/         ← LLM adapter
│   ├── pgvector/       ← Vector store adapter
│   └── http/           ← Chat endpoint, streaming SSE
└── sql/
    └── migrations/
        └── 0001_agent.sql  ← tasks, memories, tool_calls tables
                            ← requires pgvector extension
```

The agent module uses the same Module interface.
It registers tools from other modules via infra.Registry.
It reads tenant context from JWT to scope all memory and tasks.
It uses SSE for streaming responses to the UI.
